## Features

- ✅ 50+ items across Certifications, Courses, Projects, Tools & Labs
- 🔥 Daily streak tracker
- ⭐ Priority filter — shows only urgent items
- 🗺️ 4-phase roadmap (2026 → 2028)
- 🔍 Search across all tasks
- 💾 Progress saves locally on your device
- 📴 Works fully offline after first load

---

## Deploy on GitHub Pages

1. Upload all 5 files to this repository
2. Go to **Settings → Pages**
3. Under **Source** select **Deploy from a branch**
4. Branch: `main` | Folder: `/ (root)`
5. Click **Save**
6. Wait 2 minutes → site goes live at the URL above

---

## Install on Mobile (Edge / Chrome)

1. Open the live URL in Edge or Chrome on your phone
2. Tap `...` menu → **Add to phone** or **Install app**
3. App appears on home screen with shield icon
4. Opens fullscreen — no browser bar — like a native app
